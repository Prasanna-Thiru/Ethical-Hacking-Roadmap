# 1. Kali Overview
Kali Linux is a Debian‑based distribution designed for penetration testing, ethical hacking, and security auditing. It comes preloaded with hundreds of tools for tasks like network scanning, vulnerability assessment, password cracking, and digital forensics.

# 2. Install Kali Linux  
Step 1: Download ISO from the official Kali website.  
Step 2: Create bootable media (USB/DVD) using tools like Rufus or BalenaEtcher.  
Step 3: Boot from media and follow the installer prompts.  
Step 4: Choose installation type: full install, dual boot, or run in a VM (VirtualBox/VMware).  
Step 5: Complete setup with username, password, and partitioning.  

# 3. Update & Upgrade  
Keeping Kali up to date ensures you have the latest tools and patches.
Commands:  
sudo apt update  
sudo apt upgrade -y  
sudo apt dist-upgrade -y  
This fetches new package lists, upgrades installed packages, and applies distribution upgrades.

# 4. Basic Commands    
• pwd → Show current directory  
• ls → List files and folders    
• cd (dir) → Change directory    
• cp (src>) (dest) → Copy files    
• mv (src) (dest) → Move/rename files  
• rm (file) → Delete files  
• apt install (package) → Install software  
• man (command) → Show manual/help for a command
