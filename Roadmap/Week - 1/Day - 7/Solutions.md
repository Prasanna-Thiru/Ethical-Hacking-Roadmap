# Practice Day — Solutions

1. Explore Kali

- Open the Applications menu and browse categories (Information Gathering, Vulnerability Analysis, etc.). Use `man` pages and built-in tool help for learning options.

2. Practice Commands

- Practice basic Linux and networking commands: `ls`, `cd`, `man`, `ip a`, `ping`, `nmap -sV <target>`, `ss -tuln`.

3. Solve Basic Labs

- Use platforms like TryHackMe, Hack The Box (starting rooms), or local vulnerable VMs (VulnHub). Start with reconnaissance, scanning, then try simple exploits or credential discovery labs.

4. Document Learning

- Keep short lab notes in Markdown: objective, commands used, outputs (or links), and lessons learned. Example structure: Goal → Steps → Commands → Result → Notes.

Tip: repeat tasks, save commands in a `~/notes` folder, and version-control them if useful.
